# Privacy Policy

Last updated: 08.05.2026

## 1. Information Collected
This Discord bot may collect the following data:
- Discord User IDs
- Server (Guild) IDs
- Channel IDs
- Message content (only if required for commands or features)

## 2. How Data is Used
The collected data is used only for:
- providing bot functionality
- executing commands
- improving user experience

## 3. Data Storage
Data is stored only if necessary for bot features. It is not sold or shared with third parties.

## 4. Data Retention
Data is kept only as long as needed for the bot to function and may be deleted at any time.

## 5. Third Parties
No data is intentionally shared with third parties.

## 6. Security
Reasonable measures are taken to protect stored data, but no system is 100% secure.

## 7. User Rights
Users may request data deletion by contacting the bot developer.

## 8. Contact
For privacy concerns, contact the bot developer via Discord or support server.
